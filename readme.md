## Commands

### Source Environment
```bash
pixi shell -e humble
```

### Build
```bash
pixi run -e humble build
```

### Clean
```bash
pixi run -e humble clean
```

### Add package
```bash
pixi add --feature humble ros-humble-rmw-cyclonedds-cpp
```

### Manually start 
```bash
pixi shell -e humble
. install/setup.zsh
```
